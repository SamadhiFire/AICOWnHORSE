# 后端接口对接文档（P0 已实现）

适用对象：前端联调  
Base URL：`/api/v1`  
公网调试地址（Base URL）：`https://eogoylqiqymp.sealosbja.site/api/v1`  
鉴权方式：`Authorization: Bearer <token>`（除登录接口外均需）  
统一时间单位：毫秒时间戳  
统一字段风格：camelCase  

## 通用响应结构

成功：
```json
{ "code": 0, "message": "ok", "data": { } }
```

失败（示例）：
```json
{ "code": 40100, "message": "未登录或 token 失效", "data": { } }
```

错误码建议：
- 40001 参数错误
- 40100 未登录或 token 失效
- 40400 资源不存在
- 50000 服务异常

# 公网调试示例

- 游客登录：
```bash
curl -s -X POST https://eogoylqiqymp.sealosbja.site/api/v1/auth/guest \
  -H 'Content-Type: application/json' \
  -d '{"nickname":"张三","avatarUrl":"https://cdn.example.com/avatar.png"}'
```

- 获取我的信息（用上一步的 token）：
```bash
TOKEN="jwt_xxx"
curl -s https://eogoylqiqymp.sealosbja.site/api/v1/users/me \
  -H "Authorization: Bearer ${TOKEN}"
```

- 说明：游客登录/注册，返回 token 与用户信息

```json
    "refreshToken": "refresh_xxx",

      "userId": "u_123",
      "nickname": "张三",

      "createdAt": 1760000000000,
      "updatedAt": 1760000000000,
      "lastLoginAt": 1760000000000

  }
}
```
- Failure（示例）：

## 题库接口

### GET /question-bank
- 状态：未实现/计划中
- 说明：查询题库，支持主 Tab 与标签过滤、分页
- Headers：`Authorization: Bearer <token>`
- Query：
  - `mainTab`: `all` | `wrong` | `mastered`
  - `tag`: `综合` | `其他` | `<customTag>`
  - `page`, `pageSize`
- Success：
```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "list": [],
    "total": 22,
    "tagStats": [
      { "tag": "综合", "count": 22 },
      { "tag": "网络基础", "count": 6 },
      { "tag": "数据分析", "count": 8 },
      { "tag": "其他", "count": 2 }
    ]
  }
}
```
- Failure：
```json
{ "code": 40100, "message": "未登录", "data": {} }
```
- 语义约束：
  - `tag=综合`：返回当前主 Tab 下全部题目并集
  - `tag=其他`：返回不在用户标签集合中的题
  - `tag=<customTag>`：返回精确匹配该标签的题
